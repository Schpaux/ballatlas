## Phase Planning Process

1. When beginning a new phase, create `phase-N-plan.md` in this directory
2. Break the phase into 2-week sprints with specific deliverables
3. Identify research dependencies that must complete first
4. Map package and schema dependencies
5. Link to relevant ADRs

Keep `ROADMAP.md` as the source of truth for high-level status.
Use this directory for execution-level planning.
